# User-centred DER Early-stage Decision-Making Tool — Web Prototype v4

This folder is ready to host on GitHub Pages.

## Files
- `index.html` — main website
- `data/` — static source-linked lookup layers
- `assets/` — available local project logos
- `README.md` — notes

## Added in v4
- OpenStreetMap/Leaflet location map remains active.
- Added static data layers for:
  - TABULA/EPISCOPE-inspired building typology lookup
  - EU Building Stock Observatory country-context lookup
  - Eurostat degree-day climate-context lookup
  - PVGIS-informed solar-yield lookup
  - source registry
- Added “Auto-selected data context” cards in Step 1.
- The tool now reads JSON data files on GitHub Pages and falls back to embedded data if opened locally.

## Important
The JSON files are structured prototype lookup layers. Before stakeholder testing, replace or calibrate values with validated extracts from the original datasets and keep the source registry updated.
